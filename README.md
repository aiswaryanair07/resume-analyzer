# RESUME ANALYZER (Python Mini Project)
## Description
This is a simple, rule-based Resume Analyzer built using Python.
The project analyzes a resume based on a selected job role and evaluates how well the resume matches the required skills and education criteria.

This project was developed as part of my learning journey to understand Python project structuring, text processing, and logical matching.

## Objectives
- Allow the user to select a job role
- Extract skills from a resume
- Identify matched and missing skills
- Calculate a skill-matching percentage
- Check basic education eligibility
- Display clear and user-friendly feedback

## Technologies Used
- Python
- File handling
- Lists and dictionaries
- Modular project structure
- Git & GitHub

## How It Works
1.The user selects a job role
2.Resume text is read from a file
3.Text is normalized (lowercased)
4.Required skills for the selected job role are loaded
5.Skills are extracted and compared
6.Matched and missing skills are identified
7.A skill-matching percentage is calculated
8.Education eligibility is checked
9.Feedback is displayed to the user

## How to run the repository
1. Clone the repository
2. Navigate to the project folder
3. Run the program

## Note

This is a rule-based project, not a machine-learning-based ATS system.
It is intended for learning and demonstration purposes only.
