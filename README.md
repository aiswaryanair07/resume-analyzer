# RESUME ANALYZER
 
 A Resume Analyzer that evaluates a resume based on skills, education and compares it with  a target job role to identify gaps and improvement areas.

 ## Current Status 
 In development (project structure and core logic set up)

 ## Planned Features
- Skill extraction and comparison
- Education relevance analysis
- Overall resume strength score
- Suggestions to improve resume for a target role
- (Later) File upload and simple UI
